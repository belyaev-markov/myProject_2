Леонард Эйлер

Генераторы простых чисел.

«Этот общий подход позволил Эйлеру сформулировать в 1772 г. фундаментальную теорему всей теории делимости чисел – так называемую «золотую теорему» о квадратичном законе взаимности. Первое доказательство этой теоремы (и название «золотая»), как уже было сказано, дал Гаусс. Из этой теоремы посыпались, как из рога изобилия, «генераторы простых чисел», т.е. формулы, при помощи которых можно получать простые числа. Такими формулами являются: x**2 + x + 17; 2*x**2 + 29; x**2 + x + 41; x**2 – 79*x + 1601; x**2 + x + 72491 и др. Если в этих формулах последовательно подставлять 0, 1, 2, 3, …, то первая формула даст 16 простых чисел, вторая – 29, третья – 40, четвертая - 80, пятая - 4923 и т.д.»

(Цитата из книги Д.С.Фаермарка «Задача пришла с картины»).
